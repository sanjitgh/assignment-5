IMPORTENT FAQ

What is the Document Object Model (DOM)?

The DOM is a tree-structured representation of HTML, XML, and other documents. It provides a way to access and manipulate these documents programmatically.

How do you select an element from the DOM?

The DOM provides methods to select elements by ID, class name, tag name, or CSS selector. For example, you can use getElementById, getElementsByClassName, getElementsByTagName, querySelector, and querySelectorAll to find specific elements in your document.

What is event delegation in the context of the DOM, and whay is it useful?

Event delegation attaches a single event handler to a parent element, handling events that bubble up from its children. This is more efficient and simplifies code, especially for dynamic content.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate attributes, use getAttribute and setAttribute. To manipulate styles, use style property or classList.
